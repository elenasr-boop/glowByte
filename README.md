# Coal Fire Risk Frontend

Frontend приложение на **TypeScript + React**, использующее **Styled Components**, **React Router DOM** и собраное с помощью **Vite**. Проект предназначен для визуализации данных о температуре, погодных условиях и операциях угольных складов, а также для расчёта риска возгораний в отдельных штабелях.

##  Возможности

### **Главная страница (Main)**

* График прогнозируемого риска возгорания на ближайшие 3 дня.
* Список всех доступных штабелей для прогноза.
* Общая сводка по рискам.

### **История (History)**

* Исторический график температуры внутри выбранного штабеля.
* История уровня риска возгорания по дням.
* Просмотр динамики изменений.

### **Аналитика (Analytics)**

* Подробная аналитика по расчетам риска.
* Таблица успешных и ошибочных прогнозов.
* Метрики качества.

### **Загрузка данных (Uploads)**

* Поп‑ап окно для загрузки данных в формате **.csv**.
* Выбор типа импортируемых данных:

  * Температура
  * Возгорание
  * Поступление угля
  * Погодные условия

### **API**

Все данные (история, прогноз, аналитика) загружаются из внешнего API.

---

## Установка и запуск

```bash
npm install
npm run build
npm run dev
```

После запуска приложение будет доступно по адресу, предоставленному Vite (обычно `http://localhost:5173`).

## Структура проекта (кратко)

```
src/
  components/      # UI компоненты
  pages/           # Страницы: Main, History, Analytics
  modals/          # Pop-up для загрузок
  hooks/           # Кастомные хуки
  services/        # Работа с API
  styles/          # Глобальные и темовые стили
  router/          # Маршрутизация
```
